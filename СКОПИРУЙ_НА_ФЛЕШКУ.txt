================================================================================
  ПОДГОТОВКА ФЛЕШКИ ДЛЯ ОФЛАЙН-КОМПЬЮТЕРА
================================================================================

ШАГ 1: Запусти этот скрипт (на ПК с интернетом):
   
   prepare_flash_drive.bat

   Он автоматически создаст папку FLASH_DRIVE_READY со всеми файлами.

--------------------------------------------------------------------------------

ШАГ 2: Скопируй на флешку:

   ✅ Папка FLASH_DRIVE_READY (всё содержимое)
   
   ✅ Установщик Docker Desktop
      Скачать: https://www.docker.com/products/docker-desktop/
   
   ✅ Установщик Ollama
      Скачать: https://ollama.com/download

--------------------------------------------------------------------------------

ШАГ 3: На офлайн-компьютере:

   1. Установи Docker Desktop → перезагрузи ПК
   2. Установи Ollama
   3. Скопируй папку ollama_models в C:\Users\Имя\.ollama\models\
   4. Скопируй файлы из папки project в любую папку
   5. Запусти run_offline.bat
   6. Открой браузер: http://localhost:8000

--------------------------------------------------------------------------------

ЧТО БУДЕТ НА ФЛЕШКЕ:

📁 FLASH_DRIVE_READY/
├── 📁 project/
│   ├── annotator.tar         (Docker-образ с проектом)
│   ├── docker-compose.yml    (конфигурация запуска)
│   ├── Dockerfile
│   ├── requirements.txt
│   ├── config_local.py
│   ├── DEPLOYMENT.md         (полная инструкция)
│   └── QUICK_START.md        (быстрая инструкция)
│
├── 📁 ollama_models/         (твои скачанные модели)
│   ├── blobs/
│   └── manifests/
│
└── СКОПИРУЙ_НА_ФЛЕШКУ.txt    (этот файл)

--------------------------------------------------------------------------------

ЕСЛИ НЕТ DOCKER:

   Можно запустить проект без Docker:
   
   1. Установи Python 3.11
   2. pip install -r requirements.txt
   3. python run.py
