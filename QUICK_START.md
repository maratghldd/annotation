# 🚀 Быстрый старт для офлайн-компьютера

## На ПК с интернетом:

```bash
# 1. Запусти скрипт подготовки
prepare_flash_drive.bat

# 2. Скопируй папку FLASH_DRIVE_READY на флешку

# 3. Скачай установщики:
#    - Docker Desktop: https://www.docker.com/products/docker-desktop/
#    - Ollama: https://ollama.com/download
```

---

## На офлайн-компьютере:

### 1. Установи программы
- Установи **Docker Desktop** → перезагрузи ПК
- Установи **Ollama**

### 2. Перенеси модели
Скопируй папку `ollama_models` с флешки в:
```
C:\Users\Имя\.ollama\models\
```

### 3. Запусти проект
```bash
# Скопируй файлы с флешки в любую папку
# Запусти:
run_offline.bat
```

### 4. Открой браузер
```
http://localhost:8000
```

---

## ✅ Готово!

Теперь можно работать без интернета.

---

## 🔧 Если что-то не так:

| Проблема | Решение |
|----------|---------|
| Docker не запускается | Перезагрузи ПК |
| Страница не открывается | Подожди 10 секунд, обнови страницу |
| Нет моделей в списке | Проверь `ollama list` в терминале |
| Ошибка подключения | Убедись, что Ollama запущен (иконка в трее) |
